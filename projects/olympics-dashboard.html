Olympics Dashboard (Qlik Sense)

This project focused on building an interactive Qlik Sense dashboard analyzing Olympic Games data (1980–2022), enabling medal tracking, gender analysis, sport-specific drilldowns, and athlete-level insights.

Watch Demo Functional Specification Document

Objective

The objective of this project was to design an interactive Qlik Sense dashboard analyzing Olympic Games data (1980–2022).

  • Trends in medal distribution by country and sport
  • Gender performance — male vs female participation and success
  • Sport-specific breakdowns with drill-through capabilities
  • Athlete profiles with medal history
  • Age and gender dynamics across events

Technologies Used

  • Qlik Sense
  • Qlik Load Script
  • Data Modeling
  • Set Analysis
  • Data Visualization

Dataset

The dataset covered athletes, medals, countries, sports, and events (1980–2022). Preprocessing was done in Qlik’s load editor:

  • Standardized inconsistent country codes (e.g., USSR → Russia)
  • Cleaned duplicate or incomplete athlete records
  • Reshaped medal data for time-series analysis
  • Built a star-schema-like model for drill-down performance

These steps ensured smooth and responsive dashboard performance even with large datasets.

Results

The final Qlik Sense application combined medal tracking, gender analysis, sport-specific insights, and athlete-level details:

  • Dynamic medal tracking with filters for year, country, gender, and sport
  • Gender analysis highlighting increased female participation since the 1980s
  • Sport-specific trends showing shifts in dominance across athletics, swimming, and gymnastics
  • Athlete profiles with medal totals and career histories
  • Age analysis showing younger gymnasts excel vs older endurance athletes

Key Learnings

  • Mastered Qlik load scripting for data cleaning and modeling
  • Applied set analysis for advanced filtering and dynamic measures
  • Optimized performance with efficient joins and pre-aggregation
  • Designed multi-sheet dashboards with KPI summaries and drilldowns
  • Improved BI storytelling for both casual users and analysts
  • Developed strong understanding of Olympic data trends
