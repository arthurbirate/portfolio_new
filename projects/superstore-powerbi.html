Superstore Sales & Shipping Analytics – Power BI

Watch Demo Project Report

Objective

This personal project aimed to build a complete business intelligence solution using Microsoft Power BI. The goal was to analyze sales, profit, and shipping performance using a real-world dataset. The dashboard provides useful information for different users, such as executives, sales managers, and supply chain teams. The project also helped to improve skills in data analysis, visualization, and dashboard design.

Technologies Used

  • Power BI
  • DAX
  • Data Modeling
  • Data Visualization

Dataset

The project used the Superstore dataset, a sample retail dataset with several years of U.S. sales data:

  • Customer orders with dates, shipping priorities, and delivery times
  • Product categories and subcategories
  • Sales, discounts, quantities, and profit margins
  • Shipping information including modes and delays
  • Geographical data by region, state, and city

Process

  • Cleaned and organized the dataset for analysis
  • Created data models in Power BI
  • Designed dashboards for executives, sales managers, and supply chain teams
  • Created DAX measures to calculate key metrics (sales, profit margin, shipping efficiency)
  • Built interactive visuals, drill-through reports, and hierarchies for easy navigation

Results

The final dashboard provided clear insights and supported interactive data exploration:

  • Filter data by region, product category, and time period
  • Key metrics for executives: Total Sales, Profit Margin, Shipping Cost
  • Side-by-side views for high-level overview and detailed operational analysis
  • Improved decision-making for both strategic and operational tasks

Key Learnings

  • How to prepare and structure raw data for analysis
  • Designing dashboards that are clear and easy to use
  • Experience with DAX, hierarchies, bookmarks, and formatting options
  • Creating dashboards for different users, from executives to operational teams
  • Balancing usability with detailed analytical information
