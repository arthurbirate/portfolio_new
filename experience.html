Cloud & IoT Monitoring and Data Analysis

Company: Van Genechten Packaging • Location: Turnhout, Belgium • Team: DevOps • Duration: February – June

Realization Document

Description

The internship focused on ensuring business continuity by monitoring cloud infrastructure and microservices within Van Genechten Packaging’s private cloud environment, based on OpenShift and Open Kubernetes Distribution.

Monitoring was set up across hardware, network, storage, IoT devices, virtual machines, containerized applications, and Kubernetes microservices. Elasticsearch handled log aggregation and analysis, while Grafana dashboards provided real-time visualization of metrics and anomalies.

Data pipelines were implemented using Logstash to preprocess, transform, and forward logs. Machine learning techniques were applied for anomaly detection, allowing predictive insights and proactive identification of potential issues.

Tools & Technologies

  • Elasticsearch — log collection and search analytics
  • Kibana — visualizing Elasticsearch data
  • Grafana — real-time dashboards
  • Logstash — collecting, transforming, and forwarding logs
  • OpenShift / Kubernetes — orchestration and cluster metrics

Tasks

  • Developed and configured Grafana dashboards to monitor virtual machines, containers, and IoT devices
  • Processed, filtered, and enriched logs with Logstash before sending them to Elasticsearch
  • Analyzed Kubernetes metrics including pod CPU and memory usage, node health, and container performance
  • Implemented anomaly detection dashboards using Elasticsearch ML to flag unusual log patterns

Key Contributions

  • Designed Grafana dashboards for real-time cloud and IoT monitoring
  • Developed Logstash pipelines to reduce Elasticsearch load and speed up queries
  • Monitored Kubernetes clusters and recommended improvements for resource efficiency
  • Integrated ML-based anomaly detection to identify issues before they affected production
  • Documented setup and dashboards for knowledge transfer to the operations team

Achievements

  • Delivered Grafana dashboards covering critical cloud and IoT metrics
  • Optimized Logstash preprocessing, reducing Elasticsearch query times by more than 30%
  • Improved infrastructure visibility for faster incident detection and resolution
  • Gained practical expertise in enterprise observability within a private cloud environment

Personal Reflection

This internship provided a hands-on introduction to system monitoring, data analysis, and cloud observability. I gained experience building dashboards, processing logs, and interpreting infrastructure metrics using Grafana, Kibana, Elasticsearch, and Logstash.

Working independently and collaborating with the team taught me proactive problem-solving and structured data analysis. I learned to identify KPIs, detect anomalies, and translate technical insights into recommendations.

Overall, this internship reinforced my interest in data engineering and analytics, bridging theory with practical application.
